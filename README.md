### ✦ Descripción

Este repositorio contiene la plataforma web diseñada para la presentación del Trabajo de Recerca titulado **"La seguridad de la información en Internet"**. El objetivo del proyecto es analizar las vulnerabilidades comunes en la red y proponer buenas prácticas de seguridad para usuarios finales.

### ✦ Apartados del Proyecto

*   **Análisis Teórico**: Investigación detallada sobre cifrado, protocolos seguros y vectores de ataque.
*   **Interfaz Interactiva**: Navegación fluida a través de los diferentes módulos de la investigación.
*   **Resultados**: Conclusiones obtenidas tras las fases de experimentación y análisis.

### ✦ Visualización

Para visualizar el proyecto localmente, simplemente abra el archivo `index.html` en cualquier navegador web moderno.
